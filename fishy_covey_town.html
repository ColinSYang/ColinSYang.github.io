Fishy Covey Town

Summary

An extension of Covey.Town, a remote collaboration tool. The extension includes a fishing game and a leaderboard to view the top scores all time.

Developed as the final project for Northeastern's CS4530, Introduction to Software Engineering.

My Contributions

  • Integrated a MongoDB database to store high scores and enable leaderboard functionality
  • Defined engineering tasks based on self-created user stories and assigned them to sprints
  • Developed a modal menu table to display the leaderboard using react-ui

Development Information

  • Tools Used: Typescript, React, MongoDB, Netlify, Heroku, Github Actions
  • Duration: March 2023 - April 2023
  • Team Size: 4

What I Learned

  • How to use Typescript and React to create a website
  • How to use user stories to create engineering tasks
  • How to use Github Actions to create a CI/CD pipeline
