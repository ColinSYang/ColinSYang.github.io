Interactive Rubik's Cube Simulator

Summary

An interactive simulator where the user can manually scramble and solve a Rubik's cube. To rotate the cube, users press keys corresponding to Rubik's cube notation (i.e. R, L, D, etc.) on their keyboard.

Developed as the final project for Northeastern's CS4300, Computer Graphics.

My Contributions

  • Used a scene graph structure to represent the Rubik's cube
  • Dynamically reassigned parent nodes to allow for proper rotations
  • Developed an algorithm so that rotations would be consistent
  • Wrote custom shaders to render the Rubik's cube

Development Information

  • Tools Used: C++, OpenGL
  • Duration: March 2022 - April 2022
  • Team Size: 1

What I Learned

  • The importance of using a scene graph to create parent-child relationships between connected objects
  • How to use OpenGL to write custom shaders
