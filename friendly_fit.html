Friendly Fit

Summary

An Android app to connect users with others who are trying to stay active. Users can create and log custom workouts, as well as share them with their friends. When creating a profile for the app, users can upload a profile picture and set a fitness goal. The app will keep track of a user's progress towards their goal based on a user's logged workouts. Aside from collaborating on custom workouts, users can message their friends.

Developed as the final project for Northeastern's CS4520, Mobile Application Development.

My Contributions

  • Integrated a MongoDB database to store high scores and enable leaderboard functionality
  • Defined engineering tasks based on self-created user stories and assigned them to sprints
  • Developed a modal menu table to display the leaderboard using react-ui

Development Information

  • Tools Used: Android Studio (Java), Firebase Authentication, Firebase Storage, Firebase Firestore Database
  • Duration: March 2023 - April 2023
  • Team Size: 3
