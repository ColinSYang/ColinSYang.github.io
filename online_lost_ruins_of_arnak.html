Online Lost Ruins of Arnak

Summary

Adapted the board game Lost Ruins of Arnak into an online multiplayer game.

Developed as a solo personal project.

My Contributions

  • Converted the board game Lost Ruins of Arnak into an online multiplayer game using Unity's built-in multiplayer tools
  • Utilized RPC calls and networked variables to sync information across clients
  • Employed an observer pattern through C# events to trigger UI changes

Development Information

  • Tools Used: Unity, C#, Netcode for Gameobjects, Github
  • Duration: November 2023 - February 2024
  • Team Size: 1
  • Genre: Digital board game
  • Platform: PC (Windows)

What I Learned

  • How to synchronize information across clients using RPCs and networked variables
  • The difference between client authoratative and server authoritative architecture and the pros and cons of each
  • How to set up Unity Gaming Service's Lobby and Relay to allow players to connect to the same game from anywhere in the world
