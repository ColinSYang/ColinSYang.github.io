Optometrist Appointment Check-In Application

Summary

An application for patients to check-in to their appointments. This application was developed to aid a hard of hearing receptionist. After a patient checked-in, a toast notification on the receptionist's computer would provide the receptionist with the necessary information to complete the patient's check-in process.

My Contributions

  • Set up a notification system to relay relevant patient information from the application to the receptionist
  • Performed user testing to inform the design of an interface that would be easy to use for elderly people with poor eyesight.

Development Information

  • Tools Used: Python, Tkinter
  • Duration: July 2022
  • Team Size: 1

What I Learned

  • The importance of user testing in improving the user experience
  • How to use Tkinter to create basic applications
